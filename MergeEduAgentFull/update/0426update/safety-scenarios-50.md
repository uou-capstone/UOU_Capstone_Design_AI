## RPT: 리포트, 학생 데이터 격리, 초대, 교차 접근

### RPT-01 선생님은 자기 강의실의 초대 학생만 본다

### RPT-02 다른 선생님 강의실 리포트 URL 직접 접근 차단

### RPT-03 학생 계정은 리포트 화면 자체에 접근 불가
- 결과: PASS

- 단계:
  1. 학생으로 로그인한다.

### RPT-04 학생별 리포트는 선택 학생 데이터만 사용

### RPT-05 학생 선택 빠른 전환 시 늦은 응답 덮어쓰기 방지

### RPT-06 같은 lecture라도 학생별 세션 기록 분리
